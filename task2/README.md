# Version Control with Git

This project demonstrates basic Git and GitHub workflows used in DevOps environments.

## Objectives

- Initialize a Git repository
- Track project changes
- Create commits
- Work with branches
- Push code to GitHub

## Git Workflow Practiced

1. Repository initialization
2. Staging files
3. Creating commits
4. Branch creation
5. Merging branches
6. Pushing to remote repository

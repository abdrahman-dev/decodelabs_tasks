## Tasks

### Task 1 — Linux & Command Line Basics

Practiced essential Linux commands used in DevOps environments.

**Commands covered:** `pwd`, `ls`, `mkdir`, `cd`, `touch`, `echo`, `cat`, `cp`, `mv`, `rm`

---

### Task 2 — Version Control with Git

Demonstrated a complete Git workflow from init to push.

**Topics covered:** repository initialization, staging, commits, branching, merging, pushing to GitHub

---

### Task 3 — CI/CD Pipeline with GitHub Actions

Built an automated CI pipeline triggered on every push to `main`.

**Topics covered:** GitHub Actions workflow, Node.js setup, dependency installation, automated testing

---

### Task 4 — Docker Containerization

Containerized a Node.js/Express application following Docker best practices.

**Topics covered:** Dockerfile, Alpine base image, layer caching, ENTRYPOINT, `.dockerignore`, `docker build`, `docker run -d -p`
